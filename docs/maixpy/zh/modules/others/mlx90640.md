可用于无接触测温。

## 参数

* 分辨率：32x24
* 测量范围：-40°C~300°C
* 温度分辨率：0.1°C
* 测量精度：±2°C
* 重复精度：±2°C
* 响应频率：8HZ
* 工作电压：3.3~5V
* 工作电流：42mA
* 工作温度：-40°C~85°C
* 尺寸：17.27mmx33mm

## 使用方法

该模块使用串口或 I2C 进行通信，确认接线正确后，从串口即可读取到测量数据。
